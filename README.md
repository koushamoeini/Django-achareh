# Django Achareh Backend

This repository contains a Django backend implementation for the Achareh assignment (job advertisement and contractor platform). The initial commit sets up the Django project, two apps, and a basic API test.

Follow these steps to run locally:

1. Create a virtual environment and install dependencies:
```powershell
python -m venv venv; .\venv\Scripts\Activate; pip install -r requirements.txt
```
2. Run migrations and tests:
```powershell
python manage.py migrate; python manage.py test
```
